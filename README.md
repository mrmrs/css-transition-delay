# css-transition-delay

Functional CSS for transition-delay

## Filesize

| File | Size |
|------|------|
| `dist/transition-delay.css` | 1617 bytes |
| `dist/transition-delay.min.css` | 1117 bytes (231 Gzipped) |

## Install

```sh
npm install css-transition-delay
```

## Usage

### Import

```css
@import "css-transition-delay";
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/css-transition-delay/dist/transition-delay.min.css">
```

### Direct

```html
<link rel="stylesheet" href="path/to/css-transition-delay/dist/transition-delay.min.css">
```

### Responsive

Responsive variants are available for each class with the following suffixes:

| Suffix | Media Query              |
|--------|--------------------------|
| `-s`   | `min-width: 32em`        |
| `-m`   | `min-width: 48em`        |
| `-l`   | `min-width: 96em`        |

Example: `.tdl0-m` applies the property at the medium breakpoint and above.

## Building

```sh
npm run build
```

Processes `src/transition-delay.css` with [Lightning CSS](https://lightningcss.dev) and outputs to `dist/`.

- `dist/transition-delay.css` — formatted
- `dist/transition-delay.min.css` — minified
